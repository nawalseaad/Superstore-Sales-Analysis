# 📊 مشروع تحليل بيانات مبيعات Superstore

## 🎯 هدف المشروع
يهدف هذا المشروع إلى تحليل بيانات مبيعات متجر Superstore باستخدام لغة Python لاستخراج رؤى تساعد في فهم أداء المبيعات واتخاذ قرارات مبنية على البيانات.

## 🛠 الأدوات المستخدمة
- Python
- Pandas
- Matplotlib
- Google Colab

## خطوات التحليل
- استيراد البيانات.
- استكشاف البيانات والتعرف على أنواعها.
- التحقق من القيم المفقودة والصفوف المكررة.
- تحليل إجمالي ومتوسط المبيعات حسب الفئة.
- تحليل المبيعات حسب المنطقة.
- استخراج المنتجات الأعلى مبيعًا ضمن فئة Technology.
- عرض النتائج باستخدام الرسوم البيانية.

## أهم النتائج
- حققت فئة **Technology** أعلى إجمالي مبيعات.
- على الرغم من ذلك، لم تكن Technology صاحبة أكبر عدد من الطلبات، مما يشير إلى أن متوسط سعر منتجاتها أعلى من الفئات الأخرى.
- كانت منطقة **West** الأعلى في إجمالي المبيعات، بينما جاءت **South** في المرتبة الأخيرة.

## مصدر البيانات
تم استخدام مجموعة بيانات **Superstore Sales Dataset** من Kaggle لأغراض التدريب والتعلم.
